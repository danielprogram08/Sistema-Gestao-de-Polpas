# Repositório Back-End SIP (Sistema Gerenciador de Poupas).

## Diretórios de trabalho
- ### Config:
    - Contém o arquivo de configuração de controle de permissões e acessos a tipos variados de requisições: POST, GET, UPDATE, DELETE, etc.

- ### Security:
    - Contém todos os arquivos de configurações de segurança: criptografia, autenticação e autorização.

- ### Controller:
    - Contém todos os arquivos de controles de requisições.

- ### Service:
    - Contém todos os arquivos da lógica de négocios e serviços.

- ### Domain:
    - Contém todos os arquivos de modelos e DTOs.

- ### Repository:
    - Contém o arquivo de persistência de dados.
